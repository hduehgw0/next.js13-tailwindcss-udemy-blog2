# Next.js 13 + TailwindCSS Blog

## 概要

Next.js 13の最新機能とTailwindCSSを使用したブログアプリケーションの学習プロジェクトです。Udemyの講座を通じて、モダンなフロントエンド開発のベストプラクティスを学んでいます。

## 目的

- Next.js 13のApp Routerの理解と実践
- TailwindCSSによる効率的なスタイリング手法の習得
- SSG/SSR/ISRの使い分けの理解
- TypeScriptを使った型安全な開発

## 使用技術

- **Framework**: Next.js 13 (App Router)
- **Styling**: TailwindCSS
- **Language**: TypeScript
- **Database**: Supabase (予定)
- **Deployment**: Vercel

## 主な機能

- ブログ記事の一覧表示
- 記事詳細ページ (動的ルーティング)
- SSG (Static Site Generation) による高速なページ表示
- レスポンシブデザイン
- TailwindCSSによるモダンなUI

## 学習内容

このプロジェクトを通じて以下を学習しています:

- **Next.js 13の新機能**
  - App Router (従来のPages Routerからの移行)
  - Server Components と Client Components
  - データフェッチングの新しいパターン

- **TailwindCSS**
  - ユーティリティファーストのCSS設計
  - レスポンシブデザインの実装
  - カスタムコンポーネントのスタイリング

- **TypeScript**
  - 型安全なコンポーネント開発
  - Propsの型定義
  - APIレスポンスの型定義

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いて確認できます。

## デプロイ

Vercelを使用してデプロイ可能です。
